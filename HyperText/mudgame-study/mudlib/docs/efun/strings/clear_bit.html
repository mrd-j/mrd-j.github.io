### NAME

    clear_bit() - zero a bit in a bit string

### SYNOPSIS

    string clear_bit( string str, int n );

### DESCRIPTION

    Return  the  new  string where bit  is cleared in string . Note
    that the old string  is not modified.

### SEE ALSO

    set_bit(3), test_bit(3)
