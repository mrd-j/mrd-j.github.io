### NAME

    reset_eval_cost() - resets the evaluation cost remaining

### SYNOPSIS

    int reset_eval_cost();

### DESCRIPTION

    reset_eval_cost(),  resets the evaluation cost remaining to the maximum
    evaluation cost.

    Returns the new eval cost.
