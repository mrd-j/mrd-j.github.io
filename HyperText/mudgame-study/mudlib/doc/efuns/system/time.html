### NAME 名称

    time() - return the number of seconds since January 1, 1970
             返回自1970年1月1日以来的秒数

### SYNOPSIS 简介

    int time( void );

### DESCRIPTION 描述

    Return the number of seconds since midnight (GMT) January 1, 1970.
    
    返回自1970年1月1日午夜（GMT）以来的秒数。
