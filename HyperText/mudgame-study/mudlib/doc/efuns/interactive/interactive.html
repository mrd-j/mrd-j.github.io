### NAME 名称

    interactive() - detects whether or not a given object is an interactive
                    检测给定对象是否为交互式对象

### SYNOPSIS 简介

    int interactive( object ob );

### DESCRIPTION 描述

    Return non-zero if 'ob' is an interactive player. 0 will be returned if
    he is link dead.
    如果“ob”是一个交互式玩家，则返回非零。如果他是失效链接，0将被返回。
