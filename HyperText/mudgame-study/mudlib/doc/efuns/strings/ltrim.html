### NAME

    ltrim() - Remove leading whitespaces (and others)

### SYNOPSIS

    string ltrim( string str );
    string ltrim( string str, string ch);

### DESCRIPTION

    Remove all leading whitespace (or other characters 'ch') from the
    string 'str' and return a new string.

### SEE ALSO

    trim(3), rtrim(3)
