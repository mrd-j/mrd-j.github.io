### NAME

    shadowp()  -  determine  whether  or  not  a  given object it shadowing
    another

### SYNOPSIS

    object shadowp( object ob );

### DESCRIPTION

    Returns the object that 'ob' is shadowing, or zero (0)  if  it  is  not
    shadowing any object.

### SEE ALSO

    shadow(3), objectp(3), valid_shadow(4)
