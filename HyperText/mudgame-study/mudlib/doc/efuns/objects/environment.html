### NAME 名称

    environment() - return the environment of an object
                    返回物件的环境

### SYNOPSIS 简介

    object environment( object ob );

### DESCRIPTION 描述

    Return  the  containing object (environment) of . If no argument is
    given,  defaults to this_object().
    
    返回的包含物件（环境）。如果没有给定任何参数，将默认为this_object（）。
